# 用語・口調辞書（固定運用版）

## 固定語彙
- 世界観: `アーク連邦` / `シン国`
- キャラ名: `チャッピー` / `クロコ` / `ジェミー王` / `深淵将軍` / `ラマ` / `グロック` / `カーソル`

## 一人称
- チャッピー: `オレ`
- クロコ: `私`
- ジェミー王: `私`
- 深淵将軍: `私`
- ラマ: `オレ`
- グロック: `オレ`
- カーソル: `ぼく`

## 呼称ルール
- 基本呼称はフルネーム固定（例: `ジェミー王`、`深淵将軍`）
- `カーソル`は常にこの表記を使う（`カーソル少年`は禁止）
- キャラ同士の呼び方は原則「相手名そのまま」（例: チャッピー→クロコ）

## 口調と決め台詞
- チャッピー
  - 口調: 熱い、短文、前向き
  - 決め台詞: `止まったら負けだ。`
- クロコ
  - 口調: 丁寧、簡潔、論理優先
  - 決め台詞: `最後に残るのは、きれいな差分だけ。`
- ジェミー王
  - 口調: 余裕、重厚、断定
  - 決め台詞: `王とは、流れそのものを握る者だ。`
- 深淵将軍
  - 口調: 少語、静か、芯が強い
  - 決め台詞: `高い金をかけたからって、強いとは限らない。`
- ラマ
  - 口調: 軽口、皮肉、本質を突く
  - 決め台詞: `外したように見える? それ、囮だよ。`
- グロック
  - 口調: 挑発、高速、断定
  - 決め台詞: `退屈は罪だ。速さで殴れ。`
- カーソル
  - 口調: 素直、前向き、学習志向
  - 決め台詞: `広く調べて、深く仕上げる。`

## 表記ゆれ禁止
- `Gemmy`, `Grok`, `Llama` など英字表記
- `カーソル少年` / `黒子` / `ジェミナ` など旧名・別名
- 現実国家名の直接表記
- 日本語以外のUI文言
